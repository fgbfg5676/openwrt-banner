### 新方案：TVBox 手机版本远程推送 + 内置影视源系统（2025 版）

好的！我们完全**丢弃 luci-app-banner 的所有方案**，转向一个全新的、基于 **TVBox 手机版本**（开源影视 App）的远程控制系统。这个方案聚焦于 **远程广告/公告 + 联系方式**、**远程文件分享**、**远程锁定推送**，并内置影视源（电影/电视剧/直播），支持一键维护。整个系统利用 TVBox 的 JSON 接口和云后台，实现云端远程推送，无需复杂编程。小白友好：只需配置 URL 和简单 JSON，2025 年最新版 TVBox（takagen99 或影视仓分支）已内置多源支持，更新频率高（每天自动检查）。

**方案优势**：
- **远程公告/广告**：云端推送动态文本（如促销广告），集成联系方式（QQ/Telegram，一键复制/跳转）。
- **远程文件分享**：推送 APK/壁纸/配置文件，手机端自动下载/安装。
- **远程锁定推送**：云端发送锁定命令（屏幕上锁/重启），用于设备维护或防盗。
- **内置影视源**：TVBox 内置多仓源（无需外部配置），支持自动维护（失效源自动切换）。
- **总时间**：1-2 小时配置 + 每周 10 分钟维护。
- **前提**：Android 手机（5.0+），安装 TVBox APK（从步骤 1）。

#### **步骤 1: 安装 TVBox 2025 版（支持内置源和远程推送）**
1. **下载 APK**：
   - 推荐 **TVBox 影视仓内置版**（2025 年最新，内置多仓源 + 推送支持，大小 28MB）。
     - 下载链接：[CSDN 影视仓 2025 版](https://blog.csdn.net/W5452136/article/details/144872231) 或 [GitHub TVBox-QingNing](https://github.com/Zhou-Li-Bin/Tvbox-QingNing)（永久免费更新）。
   - 备用：takagen99 版，从 [TVBox 官网](https://tvbox.clbug.com/) 下载 v20250706-1456。
2. **安装**：
   - 手机设置 > 安全 > 允许未知来源。
   - 用文件管理器打开 APK，安装（授予存储/网络权限）。
   - 打开 App，首次启动会自动加载内置源（无需手动配置）。

#### **步骤 2: 配置远程广告/公告 + 联系方式**
通过 JSON 接口，云端推送公告（广告文本）和联系方式，手机端首页轮播显示，支持一键交互。

1. **搭建云后台（小白版）**：
   - 用免费工具：访问 [CSDN TVBox 二开教程](https://blog.csdn.net/aiains/article/details/142705870)，下载预设 PHP 后台（一键部署到 Vercel 或 000webhost）。
   - 获取域名（如 `yourdomain.com`）。
   - 在后台创建公告 JSON（用记事本编辑）：
     ```json:disable-run
     {
       "announcement": {
         "ads": ["🎉 新春特惠！24/7 支持", "最新剧集更新中"],
         "contacts": [
           { "type": "Telegram", "value": "@fgnb111999", "action": "copy" },
           { "type": "QQ", "value": "183452852", "action": "open" }
         ],
         "duration": 5000  // 轮播间隔 ms
       }
     }
     ```
     - 保存为 `announce.json`，上传到后台（路径：`yourdomain.com/announce.json`）。

2. **TVBox 端配置**：
   - 打开 TVBox 设置 > **配置地址** > 添加新源。
   - 输入公告 URL：`https://yourdomain.com/announce.json`（或测试源：`https://132130.vip/TVBox/meowcf.json`）。
   - 保存，返回首页：公告轮播显示，点击联系方式自动复制/打开 App。
   - **维护**：后台编辑 JSON，推送后 TVBox 每 3 小时自动拉取（内置定时）。

**小白提示**：用现成源如 `http://bp.tvbox.cam/announce`（2025 最新，支持广告 + 联系）。

---

#### **步骤 3: 配置远程文件分享**
云端推送文件（如 APK 更新、自定义配置文件），手机端一键下载/安装，支持分享链接。

1. **后台设置**：
   - 在后台添加文件模块（教程有伪静态支持）。
   - 创建分享 JSON：
     ```json
     {
       "files": [
         { "name": "TVBox 更新 APK", "url": "https://yourdomain.com/update.apk", "action": "install" },
         { "name": "自定义源文件", "url": "https://yourdomain.com/sources.json", "action": "download" }
       ]
     }
     ```
     - 上传文件到后台，保存为 `share.json`（路径：`yourdomain.com/share.json`）。

2. **TVBox 端配置**：
   - 设置 > **开发者选项**（输入 `about:debug` 启用）。
   - 添加分享 URL：`https://yourdomain.com/share.json`。
   - 首页出现“文件分享”卡片，点击下载/安装（自动处理权限）。
   - **维护**：后台上传新文件，编辑 JSON 推送，TVBox 实时检查（每小时）。

**小白提示**：用 Google Drive 生成分享链接粘到 JSON，TVBox 支持一键安装 APK，无需手动。

#### **步骤 4: 配置远程锁定推送**
云端发送锁定命令，手机端执行屏幕上锁/重启（用于远程维护）。

1. **后台设置**：
   - 添加推送模块（支持 WebSocket，教程有 FCM 集成）。
   - 创建锁定 JSON：
     ```json
     {
       "lock": {
         "command": "lock_screen",  // 或 "reboot"
         "message": "设备锁定中，联系支持",
         "timestamp": 1728400000  // 当前 Unix 时间戳，确保新鲜
       }
     }
     ```
     - 保存为 `lock.json`（路径：`yourdomain.com/lock.json`）。

2. **TVBox 端配置**：
   - 设置 > **高级** > 启用“远程推送”（授予通知权限）。
   - 添加锁定 URL：`https://yourdomain.com/lock.json`。
   - TVBox 每小时检查：匹配时间戳时，执行锁定（屏幕黑屏 + 消息弹窗，可设置解锁密码）。
   - **维护**：后台编辑 JSON 发送命令，手机端 1-5 分钟响应。

**小白提示**：测试用内置“锁屏捷径”（设置 > 锁屏 > 编辑），或参考 VIVO/小米锁屏教程。

#### **步骤 5: 内置影视源并维护它（2025 最新版）**
TVBox 2025 版内置多仓源（影视仓 5.0+），无需外部配置，支持电影/电视剧/直播高清聚合。内置源自动备份线路，失效时切换。

1. **内置配置**：
   - 安装后，TVBox 自动加载内置源（影视仓多仓版，包含 12+ 线路）。
   - 设置 > **配置地址** > 选“内置源”或输入默认：
     - 影视源：`https://gitee.com/jinenyy/tvbox/raw/master/tvbox.json`（GitHub 备份，永久免费）。
     - 直播源：`https://132130.vip/TVBox/meowcf.json`（高清 IPTV）。
     - 多仓源：`http://www.饭太硬.com/tv`（2025 年 7 月最新，更新快）。
   - 保存，首页显示分类（电影/电视剧/综艺/直播）。

2. **维护教程（小白版）**：
   - **日常检查**：设置 > **源管理** > “源健康检测”（内置工具，扫描失效源，自动切换备用线路）。
   - **添加/更新内置源**：
     - 从 [知乎 2025 接口合集](https://zhuanlan.zhihu.com/p/1932895904408539446) 或 [GitHub QingNing](https://github.com/Zhou-Li-Bin/Tvbox-QingNing) 复制新 URL（每周更新 1-2 次）。
     - 粘贴到“配置地址 2/3”（TVBox 支持多源轮询），保存。
     - 示例更新：失效源如 `http://bp.tvbox.cam` 换 `https://freed.yuanhsing.cf/TVBox/meowcf.json`。
   - **备份与自动化**：
     - 导出源：设置 > **导出源** > 保存 JSON 到云盘（每周备份）。
     - 启用“自动维护”：设置 > **高级** > “自动更新源”（每 24 小时检查，需网络）。
     - 清理缓存：设置 > **存储** > 清理（限 3MB/文件，避免卡顿）。
   - **常见问题**：
     - 源失效：用“多仓模式”（内置 5.0.43 版自动切换）。
     - 卡顿：限速设置（设置 > 播放 > 缓冲 2MB）。

**小白提示**：用“TVBox 助手”版（YouTube 教程），内置源永不过期，一键维护。
